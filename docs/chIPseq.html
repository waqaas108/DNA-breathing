Analysis of ChIP-seq data

Correlations of Motif positions and Flipping features

Our hypothesis is that flipping features inside or near motifs may determine binding affinity. To test this, we extracted all DNA sequences with motifs. Then we computed the average flipping feature across all positions in the motifs, and found the correlation between this average flipping feature and the binding affinity of the sequence for every sequence.

The ChIP-seq data used to create these results was a dataset of 886625 sequences, with each sequence having a length of 500 basepairs. To reduce the size of each sequence, 150 basepairs of flanking sequences were removed from either side of the sequence, leaving a total of 200 basepairs.

A total of 137 different cell lines and 64 TFs were represented in the ChIP-seq peaks. For each sequence, the presence of binding in each cell line was represented as a binary value. The binding affinity of the cell line (per TF) with the highest number of positive values was taken as the binding affinity of the sequence to a particular TF.

Data overview

The original ChIP-seq dataset used to create these results was a dataset of 886625 sequences, with each sequence having a length of 500 basepairs. To reduce the size of each sequence, 150 basepairs of flanking sequences were removed from either side of the sequence. To further reduce the number of sequences we had to simulate bubble data for, we filtered out sequences with positive label values for the TFs in question, making sure we had at least 3000 sequences with a positive hit for each TF. This criteria left us with 41268 sequences in total.

Bubble data

As in the gcPBM analysis, we used a bubble length threshold of 10 and an amplitude of 3.5 Armstrong. We used a probability threshold at the 95th percentile to decide if a position was positive for a bubble or not. With this threshold, 16301 out of 41268 (39.50%) of the sequences had a bubble present.

Motifs

We used the FIMO tool from the MEME suite to scan for motif presence. We used the following p-value thresholds to determine strong or weak motif matches:

Statistical Investigations

Fisherâ€™s exact test

The effect of the distance between bubbles and motifs for each sequence with both a bubble and a motif

We then investigated the effect of the distance between bubbles and motifs on binding affinity. To this end, we categorized the sequences using the label to be positive (label = 1) or negative (label = 0) sequences. We calculated the minimum distance between a motif and a bubble per sequence. We then defined close distances between motifs and bubbles using a maximum threshold, and then performed a fischerâ€™s exact test at different thresholds.

Sorting out strongest motifs using p-value and testing for the effect of the distance between bubbles and motifs

We then sorted out the strongest motifs using the p-value and calculated the distance between these motifs and the nearest bubble. If no bubble was present in the sequence, we used the length of the sequence as the distance (200). We generated histograms of the distributions of these distances for positive and negative sequences and performed a Fischerâ€™s exact test with a threshold of <200. This threshold equates to either the motif and bubble co-occurred, or the motif and a bubble were not present on the sequence together.
